- Double-click vào file build_run.bat để cài file run.exe và tự động mở file run.exe (yêu cầu phải có g++ trên PATH)
- Lưu ý:
  + Dòng chữ tạm biệt từ hệ thống hiện ra chỉ khi người dùng chạy run.exe từ build_run.bat (chạy code lần đầu)
  + Nếu chạy code từ lần 2 trở đi bằng cách mở trực tiếp file run.exe, sau khi thoát chương trình thì cửa sổ tắt luôn mà không để lại câu tạm biệt nào.
